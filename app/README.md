# ITER — Flutter Client

Flutter client for the ITER MVP. See the [repo root README](../README.md) and [srs/00-overview.md](../srs/00-overview.md) for product context.

## Stack
- Flutter stable (3.41.9)
- Maps: `maplibre_gl` + OpenFreeMap vector tiles (added in F3.1 — see [ADR-005](../ralph/DECISIONS.md))
- Auth: Supabase (added in F1.1)
- EXIF / photo upload: `exif` package (added in F2.2)
- Generated OpenAPI client: `lib/api/` (generated by `scripts/codegen.sh` after F0.5 + F0.1 both ship)

## Build

```bash
cd app
flutter pub get
flutter analyze
flutter test
flutter build apk --debug    # Android (works in WSL2)
flutter build ios            # iOS — requires macOS + Xcode + CocoaPods
```

## WSL2 Limitation
This client scaffold was created in WSL2 where iOS builds are not possible (Xcode is macOS-only). The iOS project structure exists for a future macOS contributor to verify; until then, only Android builds are guaranteed green.
